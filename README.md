# Portfolio

A modern, performance-optimized portfolio website showcasing full-stack development projects with a minimalist design aesthetic.

## Live Site

**Primary:** [Netlify deployment](https://kimaya-portfolio.netlify.app)
**Mirror:**  [portfolio-4k7sosljm-kims-projects-ae0d69a9.vercel.app](https://portfolio-4k7sosljm-kims-projects-ae0d69a9.vercel.app) 

## Tech Stack

**Core**
- React 18 with TypeScript
- Vite (build tool)
- Tailwind CSS

**Libraries & Tools**
- Framer Motion (animations)
- React Router v6 (routing)
- TanStack Query (state management)
- Radix UI & shadcn/ui (component primitives)
- Lucide React (icons)

**Deployment**
- Vercel & Netlify (continuous deployment)
- GitHub (version control)

## Features

- Responsive design optimized for all devices
- Smooth page transitions and micro-interactions
- Client-side routing with React Router
- Performance-optimized with code splitting and lazy loading
- Type-safe development with TypeScript

## Local Development

```bash
# Clone repository
git clone https://github.com/Kimaya07/Portfolio.git
cd Portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The development server runs at `http://localhost:8080`
